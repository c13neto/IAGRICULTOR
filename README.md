# IAGRICULTOR
### Escopo do projeto:
O IAgricultor é uma inteligência artificial (IA) focada no conhecimento da área de agricultura, com o objetivo de auxiliar fazendeiros no plantio e cultivo na região semiárida da Bahia. O agente será capaz de fazer recomendações, explicar técnicas, identificar problemas e melhorar o desempenho das plantações
